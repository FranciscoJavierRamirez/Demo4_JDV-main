# Imagen Hero Temporal - Confianza Legítima Post

## Estado Actual

⚠️ **Se requiere imagen definitiva**

El post está listo para publicar, pero necesita una imagen hero de 1200x630px.

## Opciones Rápidas (15 minutos)

### OPCIÓN 1: Wikipedia Commons - Contraloría ⭐ RECOMENDADA

1. Ir a: https://commons.wikimedia.org/wiki/File:Contralor%C3%ADa_General_de_la_Rep%C3%BAblica.JPG
2. Descargar la imagen en alta resolución
3. Redimensionar a 1200x630px usando:
   - **Online:** https://www.iloveimg.com/resize-image
   - **Local:** Vista previa (Mac), Paint (Windows), GIMP
4. Optimizar con: https://tinyjpg.com/
5. Guardar como: `confianza-legitima-cgr-suprema-2026.jpg` en esta carpeta

**Licencia:** CC BY-SA 4.0 (requiere atribución al final del post)

---

### OPCIÓN 2: Pexels - Edificio Institucional

1. Ir a: https://www.pexels.com/search/government%20building/
2. Buscar imagen con:
   - Arquitectura neoclásica o institucional
   - Orientación horizontal (landscape)
   - Colores neutros o azules
3. Descargar en tamaño grande
4. Redimensionar a 1200x630px
5. Guardar como: `confianza-legitima-cgr-suprema-2026.jpg`

**Licencia:** Pexels License (uso libre, sin atribución requerida)

**Imágenes sugeridas:**
- Fachada de tribunal o edificio de justicia
- Columnas neoclásicas
- Arquitectura gubernamental sobria

---

### OPCIÓN 3: Crear Placeholder con Canva (5 minutos)

**Si no se tiene tiempo para buscar foto:**

1. Ir a: https://www.canva.com/
2. Crear diseño personalizado: 1200 x 630 px
3. Usar plantilla "Social Media Header" o iniciar en blanco

**Diseño sugerido:**

```
┌─────────────────────────────────────────────────────┐
│                                                     │
│        Fondo: Azul institucional #1a365d            │
│                                                     │
│        Confianza Legítima:                          │
│        Corte Suprema vs. Contraloría                │
│                                                     │
│        [Icono de balanza dorado]                    │
│                                                     │
│        Análisis Jurídico - Febrero 2026            │
│                                                     │
└─────────────────────────────────────────────────────┘
```

**Elementos:**
- Fondo: Color sólido azul oscuro (#1a365d)
- Título: Tipografía serif (Georgia, Playfair Display)
- Subtítulo: Sans-serif (Arial, Inter)
- Icono: Balanza de justicia (buscar en elementos de Canva)
- Color del texto: Blanco y dorado (#d4a574)

4. Descargar como JPG
5. Guardar como: `confianza-legitima-cgr-suprema-2026.jpg`
